Sintaxe CSS

A sintaxe do CSS é bastante simples. Primeiro temos o seletor, que pode ser tanto uma classe, um id, ou até uma tag do HTML, depois de escolhermos nosso seletor, nós vamos abrir chaves {}, dentro dela nós colocaremos nossas declarações.

Primeiro pegamos uma propriedade(color:) e depois damos o valor (red;) seguindo do ponto e virgula para encerrar a declaração.

E assim será nossa sintaxe padrão para todos os seletores no nosso arquivo CSS.
