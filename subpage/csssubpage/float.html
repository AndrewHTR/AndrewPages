Usos do float

O float é utilizado para posicionar elementos ao lado do conteudo para estilizar a página ou fazer um design atrativo.

Por exemplo, caso tivermos uma imagem e um texto e quisermos fazer com que a imagem vá para a direita do texto, nessa hora nós vamos utilizar o elemento float para que a imagem mude de posição para a direita do texto.
