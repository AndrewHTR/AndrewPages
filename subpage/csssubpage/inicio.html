Sobre o CSS

O CSS (Cascading Style Sheets) é uma tecnologia usada para adicionar estilo (boa aparência) em uma página HTML.

O código CSS pode ser usado diretamente no arquivo HTML utilizando a tag <style> dentro do arquivo HTML, é possivel utiliza-lo na própria tag com style="", ou podemos usar em um arquivo externo atráves da tag <link>.

Com ele nós podemos deixar a página apresentavel e responsiva para os usuarios.
