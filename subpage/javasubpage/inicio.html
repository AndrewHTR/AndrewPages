Sobre o Java

Java é uma linguagem orientada a objetos que utiliza o JVM (Java Virtual Machine) para criar uma maquina virtual, fazendo com que o código possa ser facilmente portado pra qualquer dispositivo

A linguagem de programação java é feita inteiramente sobre o conceito de classes, assim sendo, um dos lemas do java é o codigo DRY (Don't Reapeat Yourself). Podemos usar a orientação a objetos para criar códigos que podem ser reutilizados ao longo do projeto
