Sobre o JavaScript

O JavaScript é uma linguagem de programação, que apesar do nome, não é nada parecida com a linguagem Java.

Nós usamos ela, geralmente, para fazer a parte mais dinâmica do site e para dar um request das informações do servidor.

JavaScript é uma linguagem de programação comum e é interpretada. Geralmente utilizamos ela juntamente com o HTML e CSS formando, assim, uma trindade. Podemos também usa-la no back-end com o Node.js
