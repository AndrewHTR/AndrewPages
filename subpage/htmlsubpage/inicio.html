Sobre o HTML

HTML (HyperText Markup Language) é uma linguagem de marcação utilizada na construção de páginas da internet. Os arquivos HTML são interpretados pelos navegadores, assim formando o texto da página, ou o bruto dela.

O HTML em si é bem simples, nós apenas utilizamos certas tags que tem seu próprio objetivo e escrevemos na página. Junto do HTML também usamos o CSS e JavaScript
