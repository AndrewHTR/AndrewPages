Semântica HTML

As Tags semânticas são tags que dizem o propósito delas para o navegador e ao programador

Elas possibilitam uma fácil percepção do objetivo ao utilizarmos a tag naquele devido lugar, além de indicar onde estará posicionada

No exemplo que vemos acima, cada tag semântica indica um local onde o texto vai estar:

  • <header> no cabeçalho da página
  • <nav> a barra de navegação
  • <section> uma seção da página
  • <article> um artigo
  • <aside> o conteudo de anúncio ou o sobre mim com menu da página
  • <footer> é o rodapé da página onde geralmente está os creditos da mesma

Além das tags semânticas exemplificadas acima, existem varias outras, porém estas são as mais comumente usadas
