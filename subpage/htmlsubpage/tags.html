Manipulação de texto

Utilizando certas tags nós podemos manipular o texto

Podemos utiliza-las para que o leitor foque naquela palavra especifica

Utilizamos tags como <strong> para colocar peso nas palavras e fazer com que o leitor preste mais atenção nela

Ou também utilizamos tags como <h1> para definirmos importância de texto

HTML em si é apenas o bruto do site(texto, botões e etc..)

Tags de Header

As tags de header são usadas comumente para definir a importância de uma sessão de texto

Indo de um nível de 1 a 6, cada nível tem seu tamanho e define a importância do texto

Sendo <h1> o mais importante e utilizado apenas, geralmente, apenas uma vez

Ele dá o titulo do site quando pesquisamos em algum mecanismo de pesquisa

Texto da página

No html nós temos tags para criar o texto da página

Coisas como parágrafos, quebra de linha e etc...

Com essas tags nós fazemos o bruto da página podendo modifica-lá depois com o uso do css

É uma boa prática sempre fazer o bruto primeiro antes do css para ter uma visão maior do que fazer

Tag Explicação
<p> Cria um parágrafo
<pre> Mantém os espaços do texto (quebra de linha e etc...)
<br> Quebra uma linha
<hr> Adiciona uma linha de ponta a ponta no site
<a> Adiciona um link a palavra ou frase especificada
<img> Coloca uma imagem no site
<blockquote> Adiciona aspas (quote) a um pedaço de texto
<q> Adiciona aspas a palavra ou frase especificada

Sobre as tags ao lado

Podemos ver que temos bastante tags para manipular o nosso texto bruto html

Devemos sempre utiliza-los para tornar o código legivel e pelo uso em si das tags

Todas elas tem seus aspectos próprios, então devemos pensar bem ao utilizar uma tag

A tag <p> e <pre> são bem parecidas na superficie, porém elas não tem o mesmo objetivo

Vantagens e descantagens, todas as tags tem alguma delas, então devo usar com sabedoria
