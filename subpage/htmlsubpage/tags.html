Manipulação de texto

Utilizando certas tags nós podemos manipular o texto

Podemos utiliza-las para que o leitor foque naquela palavra especifica

Utilizamos tags como <strong> para colocar peso nas palavras e fazer com que o leitor preste mais atenção nela

Ou também utilizamos tags como <h1> para definirmos importância de texto

HTML em si é apenas o bruto do site(texto, botões e etc..)

Tags de Header

As tags de header são usadas comumente para definir a importância de uma sessão de texto

Indo de um nível de 1 a 6, cada nível tem seu tamanho e define a importância do texto

Sendo <h1> o mais importante e utilizado apenas, geralmente, apenas uma vez

Ele dá o titulo do site quando pesquisamos em algum mecanismo de pesquisa
