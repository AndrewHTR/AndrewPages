Sobre o python

O python é uma linguagem de programação de alto nivel, interpretada e orientada a objetos. Ela encorpa basicamente tudo que é possivel fazer com a tecnologia

Com ela podemos fazer inteligencia artificial, aplicativos, back-end, etc... Conseguimos fazer bastante coisa com ela

Ela é um tanto lenta, porém com o uso certo ela tende a se tornar extremamente poderosa.
